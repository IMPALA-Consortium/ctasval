# Sample Site Data

This function samples one random site per parameter from the given data
frame.

## Usage

``` r
sample_site(df, site = "sample_site")
```

## Arguments

- df:

  Data frame containing the study data.

- site:

  The site to sample from. Default is "sample_site".

## Value

A data frame with the sampled site data.
