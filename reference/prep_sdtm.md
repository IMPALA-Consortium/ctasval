# Prepare SDTM Data

Prepare SDTM Data

## Usage

``` r
prep_sdtm(lb, vs, dm, scramble = TRUE)
```

## Arguments

- lb:

  Data frame containing the LB data.

- vs:

  Data frame containing the VS data.

- dm:

  Data frame containing the DM data.

- scramble:

  Logical indicating whether to scramble the SITEID in the DM data.
  Default is TRUE.
